# Minecraft End Update Mod

## Description
This mod adds new mobs and blocks to the Minecraft End dimension, enhancing gameplay with unique challenges and features.

## Features
- **New Mobs**:
  - **Jacal**: A hostile mob that attacks players.
  - **Kigal**: Another hostile mob that targets players.
  - **Jikal Boss**: A powerful boss mob that drops 1699 XP upon defeat.

- **New Blocks**:
  - **End Crystal Block**: A decorative block with special properties.
  - **Void Stone**: A new material block found in the End.
  - **Ethereal Glass**: A transparent block with unique aesthetics.
  - **Ender Pearl Block**: A block crafted from ender pearls.

## Building

Run the following command:
```bash
./gradlew build
```

The compiled JAR will be located in `build/libs/`

## Installation

1. Download the latest JAR from the releases
2. Place it in your `mods` folder
3. Launch Minecraft with Fabric Loader
